# 西瓜🍉宣言

0. 普遍意义下，西瓜🍉应是一种被**至少一人**认定为可以被称作“**西瓜**”的**物质**存在。
1. 除非正常情况下，西瓜🍉理应是**很好吃**的。
2. **任何人**都有**喜欢进食西瓜🍉**的权利。
3. 不论以何种方式进食西瓜🍉，只要**不是**出于**亵渎、侮辱或贬低**西瓜🍉的目的，便**应有如此进食的权利**。
